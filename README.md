Este es un proyecto personal de auto aprendizaje sobre un portafolio web para que muestre mi Curriculum Vitae, en el cual puedo acceder como Admin y modificar mi CV cuando lo ocupe y luego descargarlo en PDF.

El portafolio fue hecho en Javascript.
Conecté la cuenta de Github a un Ubuntu en (una instalación de WSL en mi PC) que va a cumplir la función de servidor Linux.
En mi PC tengo el Visual Studio, en el cual haré los cambios en la aplicación
